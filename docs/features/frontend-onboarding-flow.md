The frontend onboarding flow, documented in docs/features/frontend-onboarding-flow.md, is a four-step, user-centric process in Next.js designed to maximize conversion by balancing privacy, minimal friction, and data quality. The journey encompasses LGPD-compliant consent, lifestyle habit assessment, guided photo capture with an visual overlay, and a high-fidelity processing loader. [1, 2]
